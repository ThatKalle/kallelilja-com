# Security Policy

## Reporting a Vulnerability

To report a security issue or vulnerability, please email githubsec@kallelilja.com with a description of the issue, the steps you took to create the issue, affected versions, and, if known, mitigations for the issue. Or use the Github [Report a vulnerability](https://github.com/ThatKalle/kallelilja-com/security/advisories/new) feature.

This project follows a 90 day disclosure timeline.
